## ***Классификация комментариев клиентов***
## Пользователи могут редактировать и дополнять описания товаров, как в вики-сообществах. То есть клиенты предлагают свои правки и комментируют изменения других. Магазину нужен инструмент, который будет искать токсичные комментарии и отправлять их на модерацию. Постройте модель, которая будет классифицировать комментарии на позитивные и негативные. Для оценки модели использовать F1-меру (не меньше 0.75).
